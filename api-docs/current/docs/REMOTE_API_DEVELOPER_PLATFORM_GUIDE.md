# Remote API Developer Platform Guide

This guide described ClawPad as a developer platform around a stealth-mode network-reachable server with token enrollment. **The framing no longer matches v0.16.0**, where the Remote API is a localhost-only surface free for personal, business and commercial use, served from `127.0.0.1:8787` without authentication.

For current developer-platform material:

- **[QUICKSTART.md](QUICKSTART.md)** — get the first call working in five minutes
- **[clawpad.ai/api](https://www.clawpad.ai/api/)** — interactive portal with per-app OpenAPI specs, parameter schemas, and copy-pasteable examples for all 8,200+ operations across 309 apps
- **README.md** — pricing, license, stats

The pre-v0.16.0 platform-guide content is preserved in git history at the [file's commit log](https://github.com/yeogirlyun/ClawPad-Release/commits/main/api-docs/current/docs/REMOTE_API_DEVELOPER_PLATFORM_GUIDE.md).
